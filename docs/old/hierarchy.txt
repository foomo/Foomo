Where to find / put what resources in the filesystem.

Take care to organize user rights well, so that no read / write conflicts occur between cli and web usage.


|| Path                                 || Function / Content	                                                                                                                                                                                    ||
|| config/<runMode>                     || config directory for run modes [test | development | production] - use the API RadConfig::getConf(...) and the toolbox to access it                                                                      ||
|| config/webServerConf                 || generated web server configuration files containing the module alias mapping - include them as needed, they will be regenerated when modules are enabled or disabled                                     ||
|| logs                                 || Log directory                                                                                                                                                                                            ||
|| modules/*                            || foomo module                                                                                                                                                                                            ||
|| var                                  || application data                                                                                                                                                                                         ||
|| var/<runMode>                        || data directory for run modes [test | development | production] available via RadConfig::getVarDir()                                                                                                      ||
|| var/<runMode>/basicAuth              || basic authentication files generated with	htpasswd and used by RadBasicAuth                                                                                                                           ||
|| var/<runMode>/cache                  || application cache files - use FoomoCache, RadCacheUtils and the toolbox to access it                                                                                                                       ||
|| var/<runMode>/db                     || database files generated by RadDbFileTpl, which is a doctrine Template                                                                                                                                   ||
|| var/<runMode>/db/0                   || database files generated by RadDbFileTpl 0001 - 1000                                                                                                                                                     ||
|| var/<runMode>/db/1 [...]		|| database files generated by RadDbFileTpl 1001 - 2000, [2001-3000, ...]                                                                                                                                           ||
|| var/<runMode>/logs                   || application log directory RadConfig::getLogDir()                                                                                                                                                         ||
|| var/<runMode>/tmp                    || application temp directory RadConfig::getTempDir()                                                                                                                                                       ||
